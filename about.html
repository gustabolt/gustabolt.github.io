Sobre o JOGAR-DADOS.com

O objetivo deste website é fornecer uma ferramenta gratuita para o usuário que deseja escolher um número aleatório contido em um intervalo.

Para isso é usada a função Math.random() da linguagem JavaScript.

A imagem animada de um dado cúbico é usada para representar a escolha visualmente.
