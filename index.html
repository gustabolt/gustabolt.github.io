Sobre o glitchLevel.com

Este website fornece uma ferramenta gratuita para jogar dados online. Tem o objetivo de ser útil para jogadores de RPG, card games, jogos de tabuleiro e demais usuários que desejam lançar um dado virtualmente e de forma simples.

Inicialmente, é carregado na bandeja um dado virtual de 6 lados. Ao clicar na bandeja, ou no botão Jogar, o lançamento ocorre e, então, é exibido o resultado, que também fica registrado na caixa Histórico.

Através do menu de opções, é possível adicionar à bandeja dados de 4, 6, 8, 10 e 20 lados (todos os dados virtuais terão a aparência de um cubo). Também pode-se adicionar um dado com um número de lados à escolha (entre 2 e 999), por meio do botão com um ponto de interrogação. O número máximo de dados na bandeja é 4.

O botão Reset restaura os dados para a posição inicial, e o botão Limpar retira todos os dados da bandeja.
